# Comprehensive-Data-Science-Medical-Costs-Quality-Analytics
s is my Final Graduation Project for the Data Science course. It showcases an end-to-end analytical pipeline, applying both Regression and Classification techniques to solve real-world predictive challenges using Python.   
Key Components
1. Healthcare Cost Prediction (Regression)

Objective: Predict individual medical costs billed by health insurance. 


Methodology: Implemented Linear Regression to analyze how demographic and lifestyle factors (like BMI and smoking status) influence healthcare expenses. 


Outcome: Achieved high predictive accuracy through careful feature selection and model tuning. 

2. Quality Classification (Decision Models)

Objective: Categorize data points into distinct quality classes. 


Methodology: Leveraged Decision Tree Classifiers to build a robust model for multi-class classification. 


Visual Analysis: Utilized Confusion Matrices to evaluate and visualize model performance across all classes. 

Technical Stack

Libraries: Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn. 


Models: Linear Regression, Decision Trees. 


Tools: Jupyter Notebook for interactive development.


I used Vibe Coding to quickly build the basic structure of the project, which gave me more time to focus on "why" the data works this way and not just "how."
